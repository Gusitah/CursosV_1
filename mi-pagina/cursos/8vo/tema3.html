Páginas Web, HTML y CSS

Guía básica para entender cómo funcionan

¿Qué es una página web?

Una página web es un documento digital que forma parte de un sitio web y puede visualizarse en un navegador (como Chrome, Firefox o Edge). Generalmente está construida con tres tecnologías principales:

  • HTML (estructura)
  • CSS (estilos y diseño)
  • JavaScript (interactividad)

¿Qué es HTML?

HTML significa HyperText Markup Language. Es el lenguaje que da la estructura a una página web. Permite organizar títulos, párrafos, imágenes, enlaces y otros elementos.

Estructura básica de un documento HTML

<!DOCTYPE html>
<html lang="es">
  <head>
    <meta charset="UTF-8">
    <title>Mi primera página</title>
  </head>
  <body>
    <h1>¡Hola Mundo!</h1>
    <p>Este es un párrafo.</p>
  </body>
</html>

Etiquetas básicas en HTML

  • <h1>...</h1> → Títulos
  • <p>...</p> → Párrafos
  • <a href="...">...</a> → Enlaces
  • <img src="imagen.jpg" alt="texto"> → Imágenes
  • <ul><li></li></ul> → Listas

¿Qué es CSS?

CSS significa Cascading Style Sheets (Hojas de estilo en cascada). Se utiliza para dar diseño y apariencia a las páginas web: colores, tamaños, fuentes, márgenes, bordes, etc.

Ejemplo básico de CSS

body {
  background-color: lightblue;
  font-family: Arial, sans-serif;
}

h1 {
  color: darkblue;
  text-align: center;
}

Con este código, el fondo se vuelve azul claro, el texto usa la fuente Arial y los títulos <h1> se muestran centrados y en color azul oscuro.
