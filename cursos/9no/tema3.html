Introducción a Página Web, HTML y CSS

Explicación básica para comprender cómo funcionan

¿Qué es una página web?

Una página web es un documento accesible en Internet que se muestra en un navegador (como Chrome, Firefox o Edge). Puede contener texto, imágenes, videos, enlaces, formularios y muchos otros elementos interactivos.

Las páginas web se construyen principalmente utilizando tres tecnologías:

  • HTML: HTML (Lenguaje de Marcas de Hipertexto, del inglés HyperText Markup Language) es el componente más básico de la Web. Define el significado y la estructura del contenido web.
  • CSS: CSS es un lenguaje informático especializado en definir y cohesionar la presentación de un documento escrito en un lenguaje de marcado como HTML.
  • JavaScript: es un lenguaje de programación e scripting que se usa principalmente para hacer las páginas web interactivas y dinámicas, permitiendo desde animaciones y actualización de contenido en tiempo real hasta validación de formularios y manejo de multimedia. Es un lenguaje interpretado que se ejecuta directamente en el navegador del usuario, siendo el tercer lenguaje fundamental de la web, después de HTML (estructura) y CSS (estilo).

¿Qué es HTML?

HTML (HyperText Markup Language) es el lenguaje de marcado que define la estructura de una página web. Un elemento HTML se distingue de otro texto en un documento mediante "etiquetas", que consisten en el nombre del elemento rodeado por "<" y ">". El nombre de un elemento dentro de una etiqueta no distingue entre mayúsculas y minúsculas.

Estructura básica de un documento HTML

<!DOCTYPE html>
<html lang="es">
<head>
    <meta charset="UTF-8">
    <title>Título de la página</title>
</head>
<body>
    <h1>Encabezado principal</h1>
    <p>Este es un párrafo de ejemplo.</p>
</body>
</html>

Etiquetas HTML comunes

  • <h1>...</h1> hasta <h6>: encabezados.
  • <p>: párrafo de texto.
  • <a href="...">: enlace.
  • <img src="..." alt="...">: imagen.
  • <ul> <li>: lista desordenada.
  • <ol> <li>: lista ordenada.
  • <table>: tablas de datos.
  • <form>: formularios.

Ejemplo de HTML práctico

<h1>Mi primera página</h1>
<p>Hola, este es un párrafo de texto.</p>
<a href="https://www.google.com">Ir a Google</a>
<img src="imagen.jpg" alt="Descripción de la imagen">

¿Qué es CSS?

CSS (Cascading Style Sheets) es el lenguaje que se usa para dar estilo y diseño a los elementos de una página web. Con CSS podemos controlar colores, tamaños, fuentes, márgenes, posiciones, efectos visuales y adaptaciones a distintos dispositivos (responsive design).

Maneras de aplicar CSS

  • En línea: dentro del mismo elemento HTML.
  • Interno: dentro de la etiqueta <style> en el <head>.
  • Externo: mediante un archivo .css enlazado al HTML.

Sintaxis básica de CSS

selector {
    propiedad: valor;
}

Ejemplo de CSS

h1 {
    color: blue;
    font-size: 30px;
}

p {
    color: #333;
    line-height: 1.5;
}

img {
    width: 200px;
    border-radius: 10px;
}

Selectores comunes

  • elemento → selecciona etiquetas (p, h1, img).
  • .clase → selecciona elementos con una clase específica.
  • #id → selecciona un elemento con un ID único.
  • * → selecciona todos los elementos.

Ejemplo práctico combinando HTML y CSS

<!DOCTYPE html>
<html lang="es">
<head>
    <meta charset="UTF-8">
    <style>
        body { background: #f0f0f0; }
        h1 { color: darkred; }
        p { font-size: 18px; }
    </style>
</head>
<body>
    <h1>Bienvenido</h1>
    <p>Este es un ejemplo con CSS aplicado.</p>
</body>
</html>
