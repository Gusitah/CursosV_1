🍔 Menú Desplegable (Dropdown)

Un menú que se despliega al pasar el mouse, perfecto para navegación y opciones.

📋 Cómo funciona:

Utilizamos la pseudo-clase :hover para mostrar un contenido oculto cuando pasamos el mouse sobre el elemento padre. El contenido se posiciona de forma absoluta y usa transiciones para un efecto suave.

🏗️ HTML:

<div class="dropdown">
    <button class="dropdown-button">Menú ▼</button>
    <div class="dropdown-content">
        <a href="#">Opción 1</a>
        <a href="#">Opción 2</a>
        <a href="#">Opción 3</a>
    </div>
</div>

🎨 CSS:

.dropdown {
    position: relative;
    display: inline-block;
}

.dropdown-button {
    background: #667eea;
    color: white;
    padding: 12px 24px;
    border: none;
    border-radius: 8px;
    cursor: pointer;
}

.dropdown-content {
    position: absolute;
    top: 100%;
    left: 0;
    background: white;
    min-width: 200px;
    box-shadow: 0 8px 16px rgba(0,0,0,0.2);
    /* Estado inicial: oculto */
    opacity: 0;
    visibility: hidden;
    transform: translateY(-10px);
    transition: all 0.3s ease;
}

/* ¡La magia está aquí! */
.dropdown:hover .dropdown-content {
    opacity: 1;
    visibility: visible;
    transform: translateY(0);
}

.dropdown-content a {
    padding: 12px 16px;
    text-decoration: none;
    display: block;
    transition: background 0.3s ease;
}

.dropdown-content a:hover {
    background: #f1f1f1;
}
Usa visibility: hidden junto con opacity: 0 para evitar que el elemento oculto interfiera con otros elementos.

🎠 Slider / Carrusel

Un carrusel de contenido que permite navegar entre diferentes elementos usando radio buttons.

📋 Cómo funciona:

Utilizamos radio buttons ocultos como controladores. Cada radio button, cuando está marcado (:checked), activa una transformación CSS que mueve el contenedor. Los labels actúan como botones navegadores.

🏗️ HTML:

<div class="slider">
    <!-- Controles ocultos -->
    <input type="radio" name="slider" id="slide1" checked>
    <input type="radio" name="slider" id="slide2">
    <input type="radio" name="slider" id="slide3">
    
    <!-- Contenido del slider -->
    <div class="slider-container">
        <div class="slide">Slide 1</div>
        <div class="slide">Slide 2</div>
        <div class="slide">Slide 3</div>
    </div>
    
    <!-- Navegación -->
    <div class="slider-nav">
        <label for="slide1"></label>
        <label for="slide2"></label>
        <label for="slide3"></label>
    </div>
</div>

🎨 CSS:

.slider {
    position: relative;
    width: 100%;
    max-width: 500px;
    overflow: hidden;
    border-radius: 10px;
}

.slider-container {
    display: flex;
    transition: transform 0.5s ease;
}

.slide {
    min-width: 100%;
    height: 200px;
    /* Cada slide ocupa el 100% del ancho */
}

/* Ocultar los radio buttons */
.slider-nav input {
    display: none;
}

.slider-nav label {
    width: 12px;
    height: 12px;
    border-radius: 50%;
    background: rgba(255, 255, 255, 0.3);
    cursor: pointer;
    transition: all 0.3s ease;
}

/* ¡La lógica del slider! */
#slide1:checked ~ .slider-container {
    transform: translateX(0%);
}

#slide2:checked ~ .slider-container {
    transform: translateX(-100%);
}

#slide3:checked ~ .slider-container {
    transform: translateX(-200%);
}

/* Indicadores activos */
#slide1:checked ~ .slider-nav label[for="slide1"],
#slide2:checked ~ .slider-nav label[for="slide2"],
#slide3:checked ~ .slider-nav label[for="slide3"] {
    background: white;
    transform: scale(1.2);
}
El selector ~ (hermano general) es clave aquí. Permite que un input afecte elementos que vienen después en el HTML.

🪟 Pop-up Modal

Una ventana modal que se superpone al contenido principal, controlada por un checkbox.

📋 Cómo funciona:

Usamos un checkbox oculto como interruptor. Cuando está marcado, el overlay se hace visible. El modal está posicionado de forma fija para cubrir toda la pantalla, y usamos flexbox para centrarlo.

🏗️ HTML:

<!-- Botón para abrir -->
<label for="popup1" class="popup-trigger">Abrir Pop-up</label>

<!-- Checkbox oculto (el controlador) -->
<input type="checkbox" id="popup1" class="popup-checkbox">

<!-- El modal -->
<div class="popup-overlay">
    <div class="popup-content">
        <!-- Botón para cerrar -->
        <label for="popup1" class="popup-close">×</label>
        <h3>¡Hola!</h3>
        <p>Este es un pop-up creado solo con CSS.</p>
    </div>
</div>

🎨 CSS:

.popup-trigger {
    background: #45b7d1;
    color: white;
    padding: 12px 24px;
    border-radius: 8px;
    cursor: pointer;
}

/* Ocultar el checkbox */
.popup-checkbox {
    display: none;
}

.popup-overlay {
    position: fixed;
    top: 0;
    left: 0;
    width: 100%;
    height: 100%;
    background: rgba(0, 0, 0, 0.8);
    display: flex;
    align-items: center;
    justify-content: center;
    /* Estado inicial: oculto */
    opacity: 0;
    visibility: hidden;
    transition: all 0.3s ease;
    z-index: 1000;
}

.popup-content {
    background: white;
    padding: 30px;
    border-radius: 15px;
    max-width: 400px;
    position: relative;
    transform: scale(0.8);
    transition: transform 0.3s ease;
}

.popup-close {
    position: absolute;
    top: 15px;
    right: 20px;
    cursor: pointer;
    font-size: 24px;
}

/* ¡La magia del modal! */
.popup-checkbox:checked + .popup-overlay {
    opacity: 1;
    visibility: visible;
}

.popup-checkbox:checked + .popup-overlay .popup-content {
    transform: scale(1);
}
Usa z-index: 1000 para asegurar que el modal aparezca por encima de todo el contenido.

🖼️ Galería de Fotos

Una galería responsiva con efectos hover para mostrar imágenes de forma atractiva.

📋 Cómo funciona:

Usamos CSS Grid para crear un layout responsivo que se adapta automáticamente. Los efectos hover dan interactividad visual sin JavaScript.

🏗️ HTML:

<div class="gallery">
    <div class="gallery-item">
        <img src="imagen1.jpg" alt="Descripción 1">
    </div>
    <div class="gallery-item">
        <img src="imagen2.jpg" alt="Descripción 2">
    </div>
    <div class="gallery-item">
        <img src="imagen3.jpg" alt="Descripción 3">
    </div>
    <!-- Más elementos... -->
</div>

🎨 CSS:

.gallery {
    display: grid;
    /* Auto-fit crea columnas automáticamente */
    grid-template-columns: repeat(auto-fit, minmax(200px, 1fr));
    gap: 15px;
    margin: 20px 0;
}

.gallery-item {
    aspect-ratio: 1; /* Cuadrados perfectos */
    border-radius: 8px;
    overflow: hidden;
    cursor: pointer;
    transition: transform 0.3s ease;
}

.gallery-item:hover {
    transform: scale(1.05);
}

.gallery-item img {
    width: 100%;
    height: 100%;
    object-fit: cover; /* Mantiene proporciones */
    transition: filter 0.3s ease;
}

.gallery-item:hover img {
    filter: brightness(1.1);
}
object-fit: cover es perfecto para mantener las proporciones de las imágenes sin distorsión.

🔍 Galería con Lightbox

Galería donde las imágenes se agrandan al hacer click, creando un efecto de visualización completa.

📋 Cómo funciona:

Cada imagen tiene un radio button oculto asociado. Al hacer click en la imagen (que es un label), se marca el radio button y se muestra el lightbox correspondiente con la imagen ampliada.

🏗️ HTML:

<div class="lightbox-gallery">
    <!-- Item 1 -->
    <div class="lightbox-item">
        <label for="lightbox1">
            <img src="thumb1.jpg" alt="Miniatura 1">
        </label>
        <input type="radio" name="lightbox" id="lightbox1" class="lightbox-trigger">
        <div class="lightbox">
            <label for="close-lightbox" class="lightbox-close">×</label>
            <img src="full1.jpg" alt="Imagen completa 1">
        </div>
    </div>
    
    <!-- Item 2 -->
    <div class="lightbox-item">
        <label for="lightbox2">
            <img src="thumb2.jpg" alt="Miniatura 2">
        </label>
        <input type="radio" name="lightbox" id="lightbox2" class="lightbox-trigger">
        <div class="lightbox">
            <label for="close-lightbox" class="lightbox-close">×</label>
            <img src="full2.jpg" alt="Imagen completa 2">
        </div>
    </div>
</div>

<!-- Radio button para cerrar todos los lightbox -->
<input type="radio" name="lightbox" id="close-lightbox" checked style="display: none;">

🎨 CSS:

.lightbox-gallery {
    display: grid;
    grid-template-columns: repeat(auto-fit, minmax(150px, 1fr));
    gap: 10px;
}

.lightbox-item {
    aspect-ratio: 1;
    border-radius: 8px;
    overflow: hidden;
    cursor: pointer;
    transition: transform 0.3s ease;
}

.lightbox-item:hover {
    transform: scale(1.03);
}

.lightbox-item img {
    width: 100%;
    height: 100%;
    object-fit: cover;
}

/* Ocultar los radio buttons */
.lightbox-trigger {
    display: none;
}

.lightbox {
    position: fixed;
    top: 0;
    left: 0;
    width: 100%;
    height: 100%;
    background: rgba(0, 0, 0, 0.9);
    display: flex;
    align-items: center;
    justify-content: center;
    /* Estado inicial: oculto */
    opacity: 0;
    visibility: hidden;
    transition: all 0.3s ease;
    z-index: 1000;
}

.lightbox img {
    max-width: 90%;
    max-height: 90%;
    border-radius: 10px;
    transform: scale(0.8);
    transition: transform 0.3s ease;
}

.lightbox-close {
    position: absolute;
    top: 30px;
    right: 40px;
    color: white;
    font-size: 30px;
    cursor: pointer;
}

/* ¡La magia del lightbox! */
.lightbox-trigger:checked + .lightbox {
    opacity: 1;
    visibility: visible;
}

.lightbox-trigger:checked + .lightbox img {
    transform: scale(1);
}
Todos los radio buttons deben tener el mismo name para que solo uno pueda estar activo a la vez.

💬 Tooltip

Pequeñas ventanas de información que aparecen al pasar el mouse sobre un elemento.

📋 Cómo funciona:

Utilizamos un elemento posicionado de forma absoluta que se muestra con :hover. El pseudo-elemento ::after crea la flecha que apunta al elemento padre.

🏗️ HTML:

<div class="tooltip">
    Pasa el mouse aquí
    <span class="tooltip-text">¡Hola! Soy un tooltip</span>
</div>

🎨 CSS:

.tooltip {
    position: relative;
    display: inline-block;
    background: #007bff;
    color: white;
    padding: 10px 20px;
    border-radius: 5px;
    cursor: pointer;
}

.tooltip-text {
    position: absolute;
    bottom: 125%; /* Aparece arriba del elemento */
    left: 50%;
    transform: translateX(-50%); /* Centrado */
    background: #333;
    color: white;
    padding: 8px 12px;
    border-radius: 6px;
    font-size: 14px;
    white-space: nowrap; /* Una sola línea */
    /* Estado inicial: oculto */
    opacity: 0;
    visibility: hidden;
    transition: all 0.3s ease;
    z-index: 100;
}

/* Flecha del tooltip */
.tooltip-text::after {
    content: '';
    position: absolute;
    top: 100%;
    left: 50%;
    transform: translateX(-50%);
    border: 5px solid transparent;
    border-top-color: #333; /* Color de la flecha */
}

/* ¡La magia del tooltip! */
.tooltip:hover .tooltip-text {
    opacity: 1;
    visibility: visible;
}
Usa white-space: nowrap para evitar que el texto del tooltip se divida en múltiples líneas.

✨ Efectos de Botones Avanzados

Botones con animaciones y efectos visuales impactantes usando solo CSS.

📋 Cómo funciona:

Combinamos :hover, :active, transiciones CSS y pseudo-elementos para crear efectos llamativos. Usamos transform, box-shadow y filter para los efectos visuales.

🎨 Efecto Glow (Brillo):

.btn-glow {
    background: linear-gradient(45deg, #667eea, #764ba2);
    color: white;
    padding: 15px 30px;
    border: none;
    border-radius: 25px;
    cursor: pointer;
    transition: all 0.3s ease;
    box-shadow: 0 0 0 rgba(102, 126, 234, 0.5);
}

.btn-glow:hover {
    box-shadow: 0 0 20px rgba(102, 126, 234, 0.8);
    transform: translateY(-2px);
}

🎨 Efecto Ripple (Ondas):

.btn-ripple {
    position: relative;
    background: #45b7d1;
    color: white;
    padding: 15px 30px;
    border: none;
    border-radius: 5px;
    cursor: pointer;
    overflow: hidden;
    transition: all 0.3s ease;
}

.btn-ripple::before {
    content: '';
    position: absolute;
    top: 50%;
    left: 50%;
    width: 0;
    height: 0;
    background: rgba(255, 255, 255, 0.3);
    border-radius: 50%;
    transform: translate(-50%, -50%);
    transition: width 0.6s ease, height 0.6s ease;
}

.btn-ripple:hover::before {
    width: 300px;
    height: 300px;
}

⏳ Spinners de Carga

Animaciones de carga que indican al usuario que algo está procesándose.

🎨 Spinner Rotativo Básico:

.spinner {
    width: 40px;
    height: 40px;
    border: 4px solid #f3f3f3;
    border-top: 4px solid #007bff;
    border-radius: 50%;
    animation: spin 1s linear infinite;
}

@keyframes spin {
    0% { transform: rotate(0deg); }
    100% { transform: rotate(360deg); }
}

🎨 Spinner de Puntos:

.spinner-dots {
    display: flex;
    gap: 8px;
}

.dot {
    width: 12px;
    height: 12px;
    background: #007bff;
    border-radius: 50%;
    animation: bounce 1.4s ease-in-out infinite both;
}

.dot:nth-child(1) { animation-delay: -0.32s; }
.dot:nth-child(2) { animation-delay: -0.16s; }
.dot:nth-child(3) { animation-delay: 0s; }

@keyframes bounce {
    0%, 80%, 100% { transform: scale(0); }
    40% { transform: scale(1); }
}

🎯 Consejos y Mejores Prácticas

✅ Consejos Importantes:

  • Semántica HTML: Usa elementos HTML semánticamente correctos antes de aplicar CSS.
  • Accesibilidad: Asegúrate de que los componentes sean navegables por teclado.
  • Fallbacks: Proporciona alternativas para navegadores que no soporten ciertas características.
  • Performance: Evita animaciones en propiedades que causen reflow (como width, height).
  • Mobile First: Diseña primero para móvil y luego escala a pantallas más grandes.

📚 Propiedades CSS Clave para Interactividad:

/* Pseudo-clases importantes */
:hover          /* Al pasar el mouse */
:active         /* Al hacer click */
:focus          /* Al recibir foco */
:checked        /* Checkbox/radio marcado */
:disabled       /* Elemento deshabilitado */
:first-child    /* Primer hijo */
:last-child     /* Último hijo */
:nth-child(n)   /* Enésimo hijo */

/* Pseudo-elementos útiles */
::before        /* Contenido antes del elemento */
::after         /* Contenido después del elemento */
::first-line    /* Primera línea de texto */
::first-letter  /* Primera letra */

/* Selectores combinadores */
.parent > .child    /* Hijo directo */
.parent .descendant /* Cualquier descendiente */
.sibling + .next    /* Hermano inmediato */
.sibling ~ .general /* Hermanos generales */

/* Propiedades de transición */
transition: property duration timing-function delay;
transform: translate() rotate() scale() skew();
animation: name duration timing-function delay iteration-count direction;

/* Propiedades de posicionamiento */
position: static | relative | absolute | fixed | sticky;
z-index: number;    /* Orden de apilamiento */
overflow: visible | hidden | scroll | auto;
Experimenta combinando estas propiedades. La creatividad en CSS viene de mezclar conceptos básicos de maneras innovadoras.
